# FixPass

Le carnet de santé intelligent de tous vos objets.

## Variables d'environnement

Créez un fichier `.env.local` avec :

```
NEXT_PUBLIC_SUPABASE_URL=https://pleydrywdubuhworsurp.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=votre_cle_anon
ANTHROPIC_API_KEY=votre_cle_anthropic
```

## Déploiement Vercel

1. Pushez ce repo sur GitHub
2. Importez sur vercel.com
3. Ajoutez les variables d'environnement dans Vercel
4. Déployez !
